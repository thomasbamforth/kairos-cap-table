# Kairos Cap Table Navigator

An interactive educational tool that teaches digital health founders and venture fellows how equity is distributed and diluted from company formation through Series A funding.

Adjust the sliders to see how SAFEs, option pools, and priced rounds reshape founder ownership at each stage.

## Run locally

Open `index.html` in any modern browser — no build step required.
